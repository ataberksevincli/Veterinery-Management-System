## Hello welcome to my Veterinary Management System Project !

To briefly talk about the project, it is a system where you can register the customer and the animals belonging to this customer to the system and manage the vaccination dates, as well as register the doctor and make vaccination reservations for your animal on the days when this doctor is available.
